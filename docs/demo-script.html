Likely questions — and how to handle them

Q1
"Where does the data come from?" — Live web search via Tavily (real-time), plus structured fallback data. The platform tells you when it's using live vs. fallback in every brief.
Q2
"How accurate is it?" — As accurate as the best available public sources, synthesised by a model trained on institutional finance. The reasoning panel gives full auditability. Think of it as a first-pass analyst, not the final word.
Q3
"Can it handle other markets?" — Yes. Any emerging market — Africa, Southeast Asia, Latin America, Middle East, South Asia. Run a query live to demonstrate.
Q4
"What's the tech stack?" — Python backend with LangGraph for the agent orchestration, Google Gemini for the LLM layer, Tavily for live search, FastAPI for the API, and a vanilla JS frontend. No third-party data subscriptions required.
Q5
"Is this ready to use?" — The pipeline is fully functional today. This is a working product, not a mockup. Next steps are around data sourcing depth, user authentication, and multi-user workspaces.
